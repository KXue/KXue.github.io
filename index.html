Growth Mindset:

In hindsight, I'm don't think this design is all that great. Too many colours are required which ultimately lead to the eye pericing orange. However, I really do like that page swapping effect. Maybe if I used dark pages and a lighter font...
There's definitely a lot more work to be done. I haven't implemented the back end nor have I added PDF readers, blog readers, etc.
It seems like having the tabs swap places is a bit confusing from a UX standpoint. It looks cool and it makes sense (kinda), but maybe I should give them fixed locations so I don't have to look up and just use memory instead when clicking on tabs. But hey, I don't always take you where you wanted to go, but I always took you where you needed to be. (paraphrased from Doctor Who season 6, episode 4: The Doctor's Wife)
If you noticed the use of the golden ratio, good for you! The content width and height are 62% (~1/1.618) of the width and height of the page. The widths of all 4 tabs combined is also 62% of the width of the page (15.5% each). However, I'm not sure if any perceived improvement in aesthetics is outweighed by the loss of space.
